# Unsync Drive

Personal LAN file drive — part of the Unsync ecosystem.  
Finder-style web UI + WebDAV for native File Explorer / file manager integration.

---

## Requirements

- Node.js 18+
- An extra HDD or dedicated path for storage

---

## First-time Setup

```bash
git clone https://github.com/chukoizkie/unsync-drive
cd unsync-drive
npm install
node setup.js
```

`setup.js` will ask you for:
- Port (default: `3700`)
- Storage path (e.g. `/mnt/data/drive`)
- Username
- Password

It writes a `.env` file automatically.

---

## Run

```bash
npm start
```

Web UI → `http://aorus-server.local:3700`  
WebDAV → `http://aorus-server.local:3700/dav`

---

## Run with PM2 (recommended)

```bash
pm2 start server.js --name unsync-drive
pm2 save
pm2 startup
```

---

## Map as a Drive in Windows (File Explorer)

1. Open **File Explorer**
2. Right-click **This PC** → **Map network drive**
3. Choose a drive letter (e.g. `X:`)
4. Folder: `http://AORUS-IP:3700/dav`
5. Check **Connect using different credentials**
6. Enter your username and password
7. Click **Finish**

Drive X: will now appear in File Explorer. Drag, drop, open files directly.

> **Tip:** Get your aorus-server LAN IP with `ip a` or check your router.

---

## Map on Android

Use **Cx File Explorer** or **Solid Explorer**:
1. Add new location → WebDAV
2. Host: `http://AORUS-IP:3700/dav`
3. Username + password
4. Done — appears as a network drive in the app

## Web UI Features

- Finder-style interface (list + grid view)
- Drag and drop upload
- Chunked uploads (handles large files)
- Create folders, rename, move, delete
- Right-click context menu
- Keyboard shortcuts: `Del` to delete, `Ctrl+A` to select all, `Esc` to cancel

---

## Environment Variables

| Variable | Description |
|---|---|
| `PORT` | Port to listen on (default: 3700) |
| `STORAGE_ROOT` | Absolute path to your storage directory |
| `JWT_SECRET` | Secret key for JWT tokens (auto-generated by setup.js) |
| `DRIVE_USERNAME` | Login username |
| `DRIVE_PASSWORD` | Login password |
